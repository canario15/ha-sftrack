# SafeTrack for Home Assistant

🚜 Home Assistant integration for the SafeTrack fleet tracking platform.

## Overview

SafeTrack allows Home Assistant to integrate GPS tracking devices such as:

- Tractors
- Trucks
- Harvesters
- Agricultural machinery

using the official SafeTrack REST API.

The project focuses on providing a clean, maintainable and extensible integration following Home Assistant development best practices.
